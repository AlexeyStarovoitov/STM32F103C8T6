STM32F103 CAN Network: цикл обмена между STM32F103, подключенного через приемопередатчик sn65HVD230 и Arduino Uno, подключенной через плату MCP2515.
Скорость передачи CAN: 500 кБит/с.
Краткое описание:
-Таймер TIM2 STM32F103 с частотой 2 Гц передает по CAN на Arduino Uno команду о включении светодиода 4, которое генерирует прерывание окончания передачи.
-Обработчик прерывания передает по CAN на Arduino Uno команду о включении светодиода 5, которое генерирует второе прерывание окончания передачи.  
-Обработчик второго прерывания передает по CAN на Arduino Uno команду о включении светодиода 6, которое генерирует третье прерывание окончания передачи.  
-Обработчик третьего прерывания сбрасывает переменную, храняющую номер зажигаемого светодиода, присваеивая ей значение 4, и ждет следующее прерывание таймера.
