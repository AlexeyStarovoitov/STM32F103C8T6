STM32F103 CAN Network: цикл обмена между STM32F103, подключенного через приемопередатчик sn65HVD230 и двумя платами Arduino Uno, подключенных через плату MCP2515.
Скорость передачи CAN: 500 кБит/с.
Краткое описание:
-В can_start() происходит запуск передачи двум платам Arduino Uno команд на включение определенных светодиодов или отображении переданных данных через дисплей.
-Каждая плата Arduino Uno принимает данные, их отображает через светодиоды или на дисплее и отправляет полученные данные в виде эха в соответствующие FIFO.  
-Обработчик прерывания для каждого FIFO проверяет правильность принятого эха и отправляет новую порцию данных каждой из Arduino Uno.
-Обработчик прерывания окончания передачи сбрасывает флаги передачи.
