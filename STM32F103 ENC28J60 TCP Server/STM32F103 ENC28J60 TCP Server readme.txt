STM32F103 ENC28J60 TCP Server: TCP Server между STM32F103, подключенного через Ethernet Controller ENC28J60, и ПК.
Скорость передачи Ethernet: 10 MБит/с.
Примечение: является продолжением проекта STM32F103 ENC28J60 UDP Client.
Краткое описание:
-Через putty ПК устанавливает связь с STM32F103 по TCP-протоколу и шлет ему информацию.
-STM32F103 отображает информацию через UART и назад шлет TCP-ответ. 
-TCP-ответ от STM32F103 отображается в putty.
