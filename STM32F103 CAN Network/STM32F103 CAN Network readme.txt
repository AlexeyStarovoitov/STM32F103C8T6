STM32F103 CAN Network: цикл обмена между STM32F103, подключенного через приемопередатчик sn65HVD230 и Arduino Uno, подключенной через плату MCP2515.
Скорость передачи CAN: 500 кБит/с.
Краткое описание:
-Таймер TIM2 STM32F103 с частотой 2 Гц передает по CAN номер светодиода, который должен зажечься на Arduino Uno. 
-Arduino Uno в ответ передает номер светодиода, который должен зажечься на STM32F103.
